b tag

The b HTML element is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. This was formerly known as the Boldface element, and most browsers still draw the text in boldface.

i tag

The i HTML element is used to change the text in italic term, which are not otherwise granted special importance. The i tag is often used to indicate a technical term.

u tag

The u HTML element is used to render enclosed text with an underline, which are not otherwise granted special importance.

select tag

The select HTML element is most often used in a form to collect user input, which are not otherwise granted special importance. It represents a control which provides a menu of options.

button tag

It is used to represent a clickable button. Inside a button you can put text.
